URL Encoding Example

Original URL: https://www.example.com/search?q=coffee & tea

Encoded URL: https://www.example.com/search?q=coffee%20%26%20tea

Encoded using JavaScript:

        const originalUrl = 'https://www.example.com/search?q=coffee & tea';
        const encodedUrl = encodeURIComponent(originalUrl);
        console.log(encodedUrl);
        // Output: https%3A%2F%2Fwww.example.com%2Fsearch%3Fq%3Dcoffee%20%26%20tea
